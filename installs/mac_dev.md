### 1. install
```sh
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```


### 2. 常用命令
```
brew install wget
brew update
brew uninstall
brew search
brew list
brew upgrade
```
